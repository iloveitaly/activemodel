   .. py:method:: sample() -> T | None
                  sample(n: Literal[1]) -> T | None
                  sample(n: int) -> list[T]

      Return a random sample of rows from the current query.

      :param n: Number of rows to return. Defaults to 1.
      :type n: int
      :param Behavior:
      :param --------:
      :param - Returns a single model instance when ``n == 1`` (or ``None`` if no rows):
      :param - Returns a list[Model] when ``n > 1`` (possibly empty list when no rows):
      :param - Sampling is performed by appending an ``ORDER BY RANDOM()`` / ``func.random()``: and ``LIMIT n`` clause to the existing query target.
      :param - Keeps original query intact (does not mutate ``self.target``) so further: chaining works as expected.
